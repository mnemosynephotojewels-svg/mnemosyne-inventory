🔍 EmailJS Template Debugger

Let's find out which parameter name your template needs!

⚠️ Why You're Here

Error: "The recipients address is empty"

Cause: Your EmailJS template doesn't know which parameter contains the recipient email.

Solution: We'll test different parameter names to find which one works!

🧪 Test Different Parameter Names

Select which parameter name to test. We'll send an email using that parameter name.

📋 How to Use This Tool

  1. Select a parameter name from the list above
  2. Enter your email address in the field
  3. Click "Test Selected Parameter"
  4. Check the result:
    • ✅ Success (200) → That parameter works! Use it in your template!
    • ❌ Error (422) → Try the next parameter
  5. When you find one that works, go to your EmailJS template and put that exact parameter in the "To email" field

🎯 What to Do When You Find the Right Parameter

When you get a SUCCESS result, follow these steps:

  1. Note which parameter worked (e.g., {{email}})
  2. Go to EmailJS Dashboard: https://dashboard.emailjs.com/admin
  3. Click "Email Templates" → Open "password_reset"
  4. Find the "To email" field at the TOP
  5. Type the EXACT parameter that worked (with {{ }})
  6. Click "Save Changes"
  7. Test your Mnemosyne password reset!
