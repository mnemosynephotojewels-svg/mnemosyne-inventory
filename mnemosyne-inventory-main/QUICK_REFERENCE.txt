🐛 COMMON ISSUES
═══════════════════════════════════════════════════════════════

PROBLEM: "Database table not created yet"
SOLUTION: Run /supabase/init.sql script

PROBLEM: "Invalid API key"
SOLUTION: Check /src/app/lib/supabase.ts
         Get correct key from Supabase Dashboard

PROBLEM: "Policy already exists"
SOLUTION: Fixed! Script uses DROP POLICY IF EXISTS

PROBLEM: Data not syncing between devices
SOLUTION: 1. Run SQL script
         2. Check API key
         3. Check network connection
