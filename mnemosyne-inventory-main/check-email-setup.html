🔧 Email Setup Diagnostic

Let's fix your verification code email system step by step

1 Check if Server Function is Deployed

⏸️ Click the button below to check deployment status

⚠️ Function Not Deployed

You need to deploy the server function first. Run this command:

bash deploy-verification-code.sh

Then click "Check Deployment" again to verify.

2 Check Resend API Key Configuration

🔒 Complete Step 1 first

⚠️ API Key Not Configured

Follow these steps:

  1. Go to resend.com/api-keys
  2. Click "Create API Key"
  3. Name: Mnemosyne Production
  4. Copy the key (starts with re_)
  5. Go to Supabase Functions Settings
  6. Scroll to "Secrets" section
  7. Click "Add a new secret"
  8. Name: RESEND_API_KEY (exactly this!)
  9. Value: Paste your Resend API key
  10. Click "Save"

Get Resend API Key → Add to Supabase →

After adding the secret, click "Check API Key" again.

3 Send Test Email

🔒 Complete Step 2 first

⚠️ Use the SAME email you used to sign up for Resend (testing mode restriction)

🎉 Setup Complete!

Your verification code email system is working perfectly!
