╔═══════════════════════════════════════════════════════════════════╗
║                      TROUBLESHOOTING                              ║
╚═══════════════════════════════════════════════════════════════════╝

Problem: "Still seeing Demo Mode"
Solution:
   1. Verify secret name is: RESEND_API_KEY (exact)
   2. Check API key has no extra spaces
   3. Redeploy the Edge Function
   4. Wait 30 seconds
   5. Clear browser cache
   6. Try again

Problem: "Email not received"
Solution:
   1. Check spam/junk folder
   2. Using test domain? Must use Resend signup email
   3. Check Resend dashboard > Logs
   4. Wait a few minutes

Problem: "Invalid API Key"
Solution:
   1. Create NEW API key in Resend
   2. Copy the ENTIRE key
   3. Update in Supabase
   4. No spaces before/after
   5. Redeploy function
