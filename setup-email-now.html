Follow these steps to enable actual email sending

Your system is working perfectly, but emails are being shown on screen instead of sent. Complete the steps below to enable real email delivery to any email address.

1
Sign Up for Resend

Resend is a developer-friendly email API. It's 100% FREE for up to 3,000 emails per month (perfect for Mnemosyne!).

🚀 Sign Up at Resend.com

✅ What you get for FREE:
• 100 emails per day
• 3,000 emails per month
• No credit card required
• Professional email delivery

📧 Important: Use a real email address that you can access - you'll need to verify it!

2
Get Your API Key

After signing up and verifying your email:

  1. Log in to Resend dashboard
  2. Click on "API Keys" in the sidebar
  3. Click "Create API Key"
  4. Give it a name: Mnemosyne Password Reset
  5. Select permission: "Sending access"
  6. Click "Create"
  7. Copy the API key (starts with re_)
🔑 Go to API Keys

⚠️ Critical: Copy the ENTIRE key! It will look like: re_AbCdEfGh123456...
You won't be able to see it again after this!

This is just for your reference - you'll need to add it to Supabase in the next step
3
Add API Key to Supabase

Now add the API key to your Supabase project:

  1. Go to Supabase Dashboard
  2. Select your project: anntzpswficnoekklsdr
  3. Click "Edge Functions" in the sidebar
  4. Find and click on: make-server-a9dec19d
  5. Go to "Secrets" or "Environment Variables" tab
  6. Click "Add New Secret"
  7. Enter:
    • Name: RESEND_API_KEY
    • Value: Your API key from Step 2
  8. Click "Save"
  9. Click "Redeploy" button
🔧 Open Supabase Dashboard

💡 Tip: Make sure there are NO extra spaces before or after the API key when pasting!

⚠️ Important: After saving the secret, you MUST redeploy the Edge Function for changes to take effect!

4
Test Email Delivery

Let's verify that everything is working:

✅ What should happen:
1. Click "Test Email Delivery"
2. Wait a few seconds
3. Check your email inbox
4. You should receive a test verification code!

📝 Note: With the free Resend test domain (onboarding@resend.dev), you can only send to the email you signed up with. To send to any email, add your own domain in Resend settings.

5
Try Password Reset

Now test the full password reset flow in your Mnemosyne app:

  1. Go to your Mnemosyne login page
  2. Click "Forgot Password?"

🔧 Troubleshooting

Still seeing "Demo Mode" message

Solution:
1. Verify API key is correct in Supabase
2. Make sure secret name is exactly: RESEND_API_KEY
3. Redeploy the Edge Function
4. Wait 30 seconds for redeployment
5. Try password reset again

Email not received

Solution:
1. Check spam/junk folder
2. Verify you're using the email you signed up to Resend with
3. Check Resend dashboard > Logs for send status
4. Make sure sender email is verified in Resend
5. Wait a few minutes (can be delayed)

"Invalid API Key" error

Solution:
1. Make sure you copied the COMPLETE API key
2. Check for extra spaces before/after the key
3. Verify the key starts with: re_
4. Try creating a new API key in Resend
5. Update the secret in Supabase and redeploy
