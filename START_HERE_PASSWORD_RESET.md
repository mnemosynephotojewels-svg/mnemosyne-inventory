# 🚀 START HERE - Password Reset in 30 Seconds

## THE KEY THING YOU NEED TO KNOW:

**Use the email from your Account Settings!**

---

## Quick Steps:

### 1️⃣ Find Your Email
```
Login → Click "Account Settings" → Look at Email field
```
**Remember this email!** (probably `mnemosyne@gmail.com`)

### 2️⃣ Request Reset
```
Login page → "Forgot Password?" → Enter that SAME email → Send Code
```

### 3️⃣ Get Your Code
**Look for a BIG YELLOW BOX on screen!**

Can't see it? Press **F12** (Console) - your code is there!

### 4️⃣ Reset Password
```
Paste code → New password → Confirm → Reset → Done! ✅
```

---

## Still Not Working?

### Most Common Issue: Wrong Email ❌

**What you entered**: `myemail@gmail.com`  
**What's in Account Settings**: `mnemosyne@gmail.com`  
**Result**: No code! ❌

**Solution**: Use `mnemosyne@gmail.com` (or whatever is in Account Settings)

## How to See Your Code

The code appears in **3 places**:

1. **🟡 YELLOW ALERT BOX** - Top of the form (most visible!)
2. **💬 Toast notification** - Top-right corner of screen
3. **🖥️ Browser Console** - Press F12, look for 🔐 emoji

**At least ONE of these will show your code!**
